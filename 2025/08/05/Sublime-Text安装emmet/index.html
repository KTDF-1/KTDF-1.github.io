Sublime Text安装emmet

引言

Emmet是一款Web前端开发工具Sublime的插件，使用仿CSS选择器的语法来生成代码，提高了HTML和CSS代码编写的速度。只需按住Tab键即可把一个简写展开成HTML和CSS的代码块，大部分标签都可自动生成。

  1. 首先安装Package Control插件管理系统
    1. 打开Sublime Text：启动Sublime Text应用程序。
    2. 打开命令面板：使用快捷键 Shift + Ctrl + P 打开命令面板 。
    3. 安装Package Control插件：在命令面板中输入 Install Package Control 并按回车键。如果是第一次安装，会弹出确认框，点击确定即可 。
  2. 安装emmet插件
    1. 打开命令面板：重启后，按住快捷键Ctrl+Shift+P或者在菜单栏选择Perferences–>Package Control即可打开命令面板，如若没有反应稍等一下便会弹出
    2. 打开package：在命令面板找到package control install package，或者在命令面板输入”pcip”即这四个单词的首字母
    3. 安装emmet插件：在package的面板中搜索emmet找到后点击回车键，稍后便会自动安装。
