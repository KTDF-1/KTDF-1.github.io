Sublime Text汉化教程

2025/8/5 -Sublime Text -教程

将Sublime Text汉化

要将Sublime Text设置为中文，可以按照以下步骤进行：

  1. 打开Sublime Text：启动Sublime Text应用程序。
  2. 打开命令面板：使用快捷键 Shift + Ctrl + P 打开命令面板 。
  3. 安装Package Control插件：在命令面板中输入 Install Package Control 并按回车键。如果是第一次安装，会弹出确认框，点击确定即可 。
  4. 安装中文语言包：再次打开命令面板（Shift + Ctrl + P）。输入 Package Control: Install Package 并按回车键。在弹出的搜索框中输入 ChineseLocalizations，选择并安装 。
  5. 设置语言为简体中文：安装完成后，重启Sublime Text。点击菜单栏中的 Help -> Language -> 简体中文 。
