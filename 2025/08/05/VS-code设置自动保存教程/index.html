VS code中设置自动保存教程

  1. 首先

    打开VS code 。

  2. 然后进行以下操作

    点击“文件”–> 选择“首选项”–>点击“设置”–> 输入“after delay” –> 将Auto Save项目设置成“after delay”。

    如果在Auto Save项目的上下有关于自动保存的选项也一并勾选

  3. 最后

    关闭设置窗口即可。
